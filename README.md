🔐 Шифрование
Пароли шифруются с использованием AES с дополнительной солью:

csharp
// Алгоритм шифрования
plainText + salt → AES Encrypt → Base64 → БД
Base64 → AES Decrypt → plainText - salt
Шифрование: AES с 256-битным ключом

Соль: Уникальное значение для каждого пароля

Формат хранения: Base64 строка

📦 Формат экспорта
Файл .pme содержит JSON-структуру:

json
{
  "Version": "1.0.0",
  "List<Passwords>(): List
  {
     "Id": 0,
      "Name": "Google",
      "Url": "https://google.com",
      "Password": "myPassword123",
  }"
  "ExportDate": "2025-06-10T14:30:00"
}
